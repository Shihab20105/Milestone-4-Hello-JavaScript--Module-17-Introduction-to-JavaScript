Simple Mathematical Operations in JavaScript

Arithmetic operators are used for performing mathematical operations.

JavaScript Arithmetic Operations

Operator ------------------------ Description

( + ) --------------------------- Addition

( - ) --------------------------- Subtraction

( * ) --------------------------- Multiplication

( ** ) --------------------------- Exponentiation (ES2016)

( / ) --------------------------- Division

( % ) --------------------------- Modulus (Remainder)

( ++ ) --------------------------- Increment

( -- ) --------------------------- Decrement
